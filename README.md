# 📚 TOTVS Protheus Studies

Repositório dedicado ao estudo e à prática das tecnologias do ecossistema TOTVS, reunindo anotações, exercícios, exemplos de código e projetos desenvolvidos ao longo do aprendizado.

## 🎯 Objetivo

Centralizar e documentar minha evolução nos estudos das principais tecnologias utilizadas no ecossistema TOTVS, com foco em aprendizado contínuo, prática de programação e desenvolvimento de soluções.

## 🛠️ Tecnologias

- ADVPL
- Protheus
- Harbour
- TLPP
- Git & GitHub
- Visual Studio Code
