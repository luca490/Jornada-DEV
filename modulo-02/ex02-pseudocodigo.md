## c. Encontrar o maior entre três números

Início
    Leia num1
    Leia num2
    Leia num3
    maior ← num1
    Se num2 > maior Então
        maior ← num2
    FimSe

    Se num3 > maior Então
        maior ← num3
    FimSe

    Escreva "O maior número é: ", maior
Fim
