# Exercício 1
--------------
### a. Some dois números.

**Resposta:** Não é um algoritmo válido.
**Justificativa:**
A instrução é ambígua, pois não conseguimos deduzir quais números devem ser somados nem como eles serão obtidos.

--------------------------------------------------------------------------------
### b. Leia um número. Se for positivo, mostre 'positivo'. Senão, mostre 'negativo'.
**Resposta:** É um algoritmo válido.
**Justificativa:** Ele não possui ambiguidades, sendo ordenado por passo a passo e finito, pois termina com uma mensagem após uma condição.

--------------------------------------------------------------------------------
### c. Fique repetindo até acabar.
**Resposta:** Não é um algoritmo válido.
**Justificativa:** Não é finito, pois nada mostra uma parada após a repetição. Além disso, a expressão "até acabar" é ambígua pois não sabe-se o passo a passo.

--------------------------------------------------------------------------------
### d. Pegue um ingrediente qualquer e cozinhe.
**Resposta:** Não é um algoritmo válido.
**Justificativa:** A instrução é ambígua, pois não esclarece o que será obtido e como será cozinhado.
