# Exercício 3
--------------
## Problema
Uma loja dá desconto de 10% para compras acima de R$ 100,00. 
Leia o valor da compra e mostre o valor final a pagar.

------------------------------------------------------------
## Estrutura do Fluxograma
1. Início
2. Entrada de dados
   - Ler o valor da compra.
3. Decisão
   - Valor da compra é maior que R$ 100,00?

### Caso "Sim":
4. Calcular o desconto:
   - "valorFinal ← valor - (valor * 0.10)"
5. Mostrar:
   - "Valor final: R$", valorFinal"
6. Fim

### Caso "Não"
4. "valorFinal ← valor"
5. Mostrar:
   - "Valor final: R$", valorFinal"
6. Fim
